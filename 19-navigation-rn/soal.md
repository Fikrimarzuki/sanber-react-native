2. Implementasi React Navigation
Pada bagian ini, Anda diminta untuk mengimplementasikan React Navigation pada aplikasi yang telah Anda bangun mulai dari Tugas 13 dan Tugas 14.

Buatlah sebuah folder “TugasNavigation” di dalam folder “Tugas” masukkan LoginScreen.js, AboutScreen.js, dan SkillScreen.js.
Tambahkan index.js yang akan digunakan sebagai letak konfigurasi React Navigation seperti pada tutorial sebelumnya.
Tambahkan file ProjectScreen.js dan AddScreen.js yang berisi komponen yang hanya menampilkan teks “Halaman Proyek” dan “Halaman Tambah”.
Buat stuktur navigasi/halaman sebagai berikut:

https://blog.sanbercode.com/wp-content/uploads/2020/07/Tugas15-1.png
Ketika membuat konfigurasi navigation di index.js, boleh menggunakan Function Component mengikuti video tutorial.
